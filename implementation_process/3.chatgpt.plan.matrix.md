对，当前这个：

```cpp
std::vector<std::vector<mpq_class>> matrix_;  // m x n
```

不适合作为最终实现。它的问题不是“小优化一下”能解决的，而是数据结构方向本身不对。SMT 里的 LRA/LIA tableau 通常非常稀疏，而且 `mpq_class` 很重，dense `m*n` 会同时浪费：

```text
1. 内存：绝大多数位置是 0；
2. update 时间：update(xj) 不应该扫所有 row，只该扫含 xj 的 row；
3. pivot 时间：pivot 不应该扫全矩阵，只该更新含 entering variable 的 row；
4. cache：vector<vector<mpq_class>> 大量小对象，局部性差；
5. 有理数运算：0 也被当成 mpq_class 存储，代价很高。
```

你最终应该把 GeneralSimplex 改成：

```text
Sparse Indexed Explicit Tableau
= 稀疏 row storage
+ 稀疏 column index
+ O(1) coefficient update/erase
+ pivot 只更新 affected rows
+ update(beta) 只更新 affected basic variables
+ violation queue
+ exact rational fast path
```

不要简单用：

```cpp
std::unordered_map<int, mpq_class>
```

做每一行。那会导致 hash 开销、迭代不稳定、pivot 中大量插删变慢。更好的最终设计是：**row 和 column 双向索引，entry 之间保存 cross-position handle**。

# SparseTableau 的核心实现

## `setCoeff(row, col, value)`

这是整个稀疏 tableau 的核心。必须同时维护 row 和 column。

```cpp
void SparseTableau::setCoeff(int r, int c, const Rat& v) {
    auto& row = rows[r];

    int rowPos = -1;
    for (int i = 0; i < static_cast<int>(row.entries.size()); ++i) {
        if (row.entries[i].col == c) {
            rowPos = i;
            break;
        }
    }

    if (rowPos != -1) {
        if (v == 0) {
            eraseCoeff(r, c);
        } else {
            row.entries[rowPos].coeff = v;
        }
        return;
    }

    if (v == 0) return;

    auto& col = cols[c];

    int newRowPos = static_cast<int>(row.entries.size());
    int newColPos = static_cast<int>(col.entries.size());

    row.entries.push_back(RowEntry{
        .col = c,
        .coeff = v,
        .colPos = newColPos
    });

    col.entries.push_back(ColEntry{
        .row = r,
        .rowPos = newRowPos
    });
}
```

上面这个版本为了清楚用了线性查找。最终实现应该给 row 再加一个轻量索引，避免 pivot 中频繁查找。

更好的 row 结构：

```cpp
struct SparseRow {
    int basicVar;
    Rat rhs;
    std::vector<RowEntry> entries;

    // col -> position in entries
    // 小行不用 hash，大行才建 index
    std::unordered_map<int, int> pos;
    bool hasIndex = false;
};
```

最终 `findRowPos`：

```cpp
int findRowPos(SparseRow& row, int col) {
    if (row.hasIndex) {
        auto it = row.pos.find(col);
        return it == row.pos.end() ? -1 : it->second;
    }

    for (int i = 0; i < static_cast<int>(row.entries.size()); ++i) {
        if (row.entries[i].col == col) return i;
    }

    return -1;
}
```

当 row 很大时自动建索引：

```cpp
void maybeBuildRowIndex(SparseRow& row) {
    constexpr int INDEX_THRESHOLD = 32;

    if (row.hasIndex || row.entries.size() < INDEX_THRESHOLD) return;

    row.pos.clear();
    for (int i = 0; i < static_cast<int>(row.entries.size()); ++i) {
        row.pos[row.entries[i].col] = i;
    }
    row.hasIndex = true;
}
```

## `eraseCoeff(row, col)`

用 swap-pop，保证 O(1) 删除，并修复 cross handle。

```cpp
void SparseTableau::eraseCoeff(int r, int c) {
    auto& row = rows[r];

    int rowPos = findRowPos(row, c);
    if (rowPos == -1) return;

    int colPos = row.entries[rowPos].colPos;
    auto& col = cols[c];

    int lastRowPos = static_cast<int>(row.entries.size()) - 1;
    int lastColPos = static_cast<int>(col.entries.size()) - 1;

    // Remove from row by swap-pop
    if (rowPos != lastRowPos) {
        RowEntry moved = row.entries[lastRowPos];
        row.entries[rowPos] = moved;

        // Fix column back-pointer of moved entry
        cols[moved.col].entries[moved.colPos].rowPos = rowPos;

        if (row.hasIndex) {
            row.pos[moved.col] = rowPos;
        }
    }

    if (row.hasIndex) {
        row.pos.erase(c);
    }

    row.entries.pop_back();

    // Remove from column by swap-pop
    if (colPos != lastColPos) {
        ColEntry moved = col.entries[lastColPos];
        col.entries[colPos] = moved;

        // Fix row back-pointer of moved entry
        rows[moved.row].entries[moved.rowPos].colPos = colPos;
    }

    col.entries.pop_back();
}
```

## `addCoeff(row, col, delta)`

```cpp
void SparseTableau::addCoeff(int r, int c, const Rat& delta) {
    if (delta == 0) return;

    Rat old = getCoeff(r, c);
    Rat nv = old + delta;

    setCoeff(r, c, nv);
}
```

# 行语义必须固定

最终 row 语义只保留这一种：

```text
row r:
  x_b = rhs[r] + Σ a[r][j] * x_j

其中：
  x_b 是 basic variable
  x_j 必须是 non-basic variable
```

不要再混用：

```text
x_b - Σ a_j x_j = rhs
```

所有公式、pivot、update、conflict 都按这个语义写。

# 添加约束时必须处理 basic 变量替换

这是很多实现会漏的关键点。

假设你要注册新线性形式：

```text
s = c + a1*x1 + ... + an*xn
```

如果当前 `x1` 已经是 basic variable，它不能直接出现在新 row 的 entries 里。因为 tableau row 只能包含 non-basic variables。

所以添加新 row 时必须先把表达式改写到当前 non-basic 变量上。

```cpp
struct LinearForm {
    Rat constant;
    std::vector<std::pair<int, Rat>> terms; // var, coeff
};
```

归约函数：

```cpp
LinearForm GeneralSimplex::rewriteToNonBasic(const LinearForm& input) {
    LinearForm out;
    out.constant = input.constant;

    std::unordered_map<int, Rat> coeffs;

    auto addTerm = [&](int v, const Rat& a) {
        if (a == 0) return;
        coeffs[v] += a;
        if (coeffs[v] == 0) {
            coeffs.erase(v);
        }
    };

    for (auto [v, a] : input.terms) {
        int row = vars_[v].basicRow;

        if (row == -1) {
            addTerm(v, a);
        } else {
            // v is basic:
            // v = rhs + Σ coeff_j * x_j
            const SparseRow& br = tab_.rows[row];

            out.constant += a * br.rhs;

            for (const RowEntry& e : br.entries) {
                addTerm(e.col, a * e.coeff);
            }
        }
    }

    out.terms.clear();
    out.terms.reserve(coeffs.size());

    for (auto& [v, a] : coeffs) {
        if (a != 0) out.terms.push_back({v, a});
    }

    std::sort(out.terms.begin(), out.terms.end(),
              [](auto& p, auto& q) { return p.first < q.first; });

    return out;
}
```

注册新 linear form：

```cpp
int GeneralSimplex::addLinearFormAsBasicAux(
    const std::string& auxName,
    const LinearForm& raw
) {
    int s = addVar(auxName);

    LinearForm f = rewriteToNonBasic(raw);

    int row = tab_.addRow(s, f.constant);

    vars_[s].basicRow = row;
    basicVars_.push_back(s);

    for (auto [v, a] : f.terms) {
        assert(vars_[v].basicRow == -1);
        tab_.setCoeff(row, v, a);
    }

    betaDirty_ = true;
    return s;
}
```

这个设计可以支持：

```text
tableau 已经 pivot 很多次后，再动态注册新 atom。
```

如果不做这个，动态 atom / disequality split / cut lemma 生成后很容易破坏 tableau invariant。

# `update(xj, v)` 必须用 column index

现在 dense 版大概率是：

```cpp
for (int row = 0; row < m; ++row) {
    beta[basicVars[row]] += matrix[row][xj] * delta;
}
```

最终应该是：

```cpp
void GeneralSimplex::updateNonBasic(int xj, const DeltaRat& v) {
    assert(vars_[xj].basicRow == -1);

    DeltaRat delta = v - vars_[xj].beta;
    if (delta.isZero()) return;

    // Only rows containing xj are affected.
    const auto affected = tab_.colEntries(xj); // copy or stable snapshot

    for (const ColEntry& ce : affected) {
        int row = ce.row;
        int xb = tab_.rows[row].basicVar;

        Rat a = tab_.rows[row].entries[ce.rowPos].coeff;
        vars_[xb].beta += DeltaRat(a) * delta;

        refreshViolationStatus(xb);
    }

    vars_[xj].beta = v;
}
```

这里复杂度从：

```text
O(number_of_rows)
```

变成：

```text
O(number_of_rows_containing_xj)
```

这是 sparse tableau 的最大收益之一。

# violation queue

不要每次 `check()` 都扫所有 basic variables 找 violated basic。

维护一个队列：

```cpp
void GeneralSimplex::refreshViolationStatus(int x) {
    if (vars_[x].basicRow == -1) return;

    bool violated = violatesLower(x) || violatesUpper(x);

    if (violated && !inViolationQueue_[x]) {
        violatedQueue_.push_back(x);
        inViolationQueue_[x] = true;
    }
}
```

取 violated basic：

```cpp
int GeneralSimplex::pickViolatedBasic() {
    while (!violatedQueue_.empty()) {
        int x = violatedQueue_.front();
        violatedQueue_.pop_front();
        inViolationQueue_[x] = false;

        if (vars_[x].basicRow != -1 &&
            (violatesLower(x) || violatesUpper(x))) {
            return x;
        }
    }

    // fallback scan, mainly after recomputeBeta()
    for (int xb : basicVars_) {
        if (violatesLower(xb) || violatesUpper(xb)) {
            return xb;
        }
    }

    return -1;
}
```

`recomputeBeta()` 后批量重建队列：

```cpp
void GeneralSimplex::rebuildViolationQueue() {
    violatedQueue_.clear();
    std::fill(inViolationQueue_.begin(), inViolationQueue_.end(), false);

    for (int xb : basicVars_) {
        if (violatesLower(xb) || violatesUpper(xb)) {
            violatedQueue_.push_back(xb);
            inViolationQueue_[xb] = true;
        }
    }
}
```

# bound 比较必须处理 infinite

```cpp
bool GeneralSimplex::violatesLower(int x) const {
    const auto& l = vars_[x].lower.bound;
    return l.isFinite() && vars_[x].beta < l.value;
}

bool GeneralSimplex::violatesUpper(int x) const {
    const auto& u = vars_[x].upper.bound;
    return u.isFinite() && vars_[x].beta > u.value;
}

bool GeneralSimplex::canIncrease(int x) const {
    const auto& u = vars_[x].upper.bound;
    return !u.isFinite() || vars_[x].beta < u.value;
}

bool GeneralSimplex::canDecrease(int x) const {
    const auto& l = vars_[x].lower.bound;
    return !l.isFinite() || vars_[x].beta > l.value;
}

bool GeneralSimplex::atUpper(int x) const {
    const auto& u = vars_[x].upper.bound;
    return u.isFinite() && vars_[x].beta == u.value;
}

bool GeneralSimplex::atLower(int x) const {
    const auto& l = vars_[x].lower.bound;
    return l.isFinite() && vars_[x].beta == l.value;
}
```

# entering variable 选择

对于 lower violation：

```text
x_i 太小，需要增大 x_i。
row:
  x_i = rhs + Σ a_j x_j
```

如果：

```text
a_j > 0
```

增大 `x_j` 可以增大 `x_i`。

如果：

```text
a_j < 0
```

减小 `x_j` 可以增大 `x_i`。

最终实现：

```cpp
int GeneralSimplex::findEnteringToIncrease(int xi) {
    int row = vars_[xi].basicRow;
    assert(row != -1);

    int best = -1;

    for (const RowEntry& e : tab_.rows[row].entries) {
        int xj = e.col;
        const Rat& a = e.coeff;

        bool eligible =
            (a > 0 && canIncrease(xj)) ||
            (a < 0 && canDecrease(xj));

        if (!eligible) continue;

        if (best == -1 || xj < best) {
            best = xj; // Bland's rule
        }
    }

    return best;
}
```

对于 upper violation：

```cpp
int GeneralSimplex::findEnteringToDecrease(int xi) {
    int row = vars_[xi].basicRow;
    assert(row != -1);

    int best = -1;

    for (const RowEntry& e : tab_.rows[row].entries) {
        int xj = e.col;
        const Rat& a = e.coeff;

        bool eligible =
            (a < 0 && canIncrease(xj)) ||
            (a > 0 && canDecrease(xj));

        if (!eligible) continue;

        if (best == -1 || xj < best) {
            best = xj;
        }
    }

    return best;
}
```

复杂度：

```text
O(nnz(row))
```

不是：

```text
O(numCols)
```

# pivotAndUpdate

```cpp
void GeneralSimplex::pivotAndUpdate(int xi, int xj, const DeltaRat& target) {
    int r = vars_[xi].basicRow;
    assert(r != -1);
    assert(vars_[xj].basicRow == -1);

    Rat aij = tab_.getCoeff(r, xj);
    assert(aij != 0);

    DeltaRat theta = (target - vars_[xi].beta) / aij;

    // Update beta using sparse column of xj
    const auto affected = tab_.colEntries(xj); // snapshot

    for (const ColEntry& ce : affected) {
        int row = ce.row;
        int xb = tab_.rows[row].basicVar;

        Rat a = tab_.rows[row].entries[ce.rowPos].coeff;

        if (xb == xi) {
            vars_[xb].beta = target;
        } else {
            vars_[xb].beta += DeltaRat(a) * theta;
        }

        refreshViolationStatus(xb);
    }

    vars_[xj].beta += theta;

    pivot(r, xi, xj);

    refreshViolationStatus(xi); // now non-basic, harmless check ignored
    refreshViolationStatus(xj); // now basic
}
```

注意：`affected` 必须是 snapshot。因为 `pivot()` 会修改 column `xj`。

# 稀疏 pivot 公式

当前 pivot row：

```text
xi = rhs + a*xj + Σ b_k*x_k
```

其中 `xi` 是 leaving basic，`xj` 是 entering non-basic。

解出 `xj`：

```text
xj = (-rhs/a) + (1/a)*xi + Σ (-b_k/a)*x_k
```

pivot 后：

```text
xj 是 basic；
xi 变成 non-basic。
```

新 pivot row：

```text
xj = rhs' + coeff(xi)*xi + Σ coeff(k)*x_k
```

其中：

```text
rhs' = -rhs / a
coeff(xi) = 1 / a
coeff(k) = -b_k / a
```

其他行如果包含 `xj`：

```text
xb = rhs_b + c*xj + Σ d_k*x_k
```

替换为新 pivot row：

```text
xb = rhs_b + c*rhs' + c*coeff(xi)*xi + Σ (d_k + c*coeff(k))*x_k
```

实现：

```cpp
void GeneralSimplex::pivot(int r, int leaving, int entering) {
    SparseRow oldPivot = tab_.rows[r]; // copy
    Rat a = tab_.getCoeff(r, entering);
    assert(a != 0);

    // Build new pivot row: entering = -rhs/a + (1/a)*leaving + Σ(-coeff/a)*x
    std::vector<std::pair<int, Rat>> newEntries;
    newEntries.reserve(oldPivot.entries.size());

    Rat newRhs = -oldPivot.rhs / a;

    // leaving becomes non-basic
    newEntries.push_back({leaving, Rat(1) / a});

    for (const RowEntry& e : oldPivot.entries) {
        int col = e.col;
        if (col == entering) continue;

        Rat coeff = -e.coeff / a;
        if (coeff != 0) {
            newEntries.push_back({col, coeff});
        }
    }

    // Take snapshot of rows containing entering before modifying column.
    std::vector<ColEntry> affected = tab_.cols[entering].entries;

    // Replace pivot row first or later? Safer:
    // remove entering from all non-pivot rows first, then replace pivot row.
    for (const ColEntry& ce : affected) {
        int row = ce.row;
        if (row == r) continue;

        Rat c = tab_.getCoeff(row, entering);
        if (c == 0) continue;

        tab_.eraseCoeff(row, entering);

        tab_.rows[row].rhs += c * newRhs;

        for (auto [col, coeff] : newEntries) {
            tab_.addCoeff(row, col, c * coeff);
        }

        tab_.rows[row].version++;

        int xb = tab_.rows[row].basicVar;
        refreshViolationStatus(xb);
    }

    // Replace pivot row.
    tab_.replaceRow(r, entering, newRhs, newEntries);

    // Update basis metadata.
    vars_[leaving].basicRow = -1;
    vars_[entering].basicRow = r;

    tab_.rows[r].basicVar = entering;
    basicVars_[r] = entering;

    // nonBasicVars_ update can be lazy or O(1) with position arrays.
    markBasicSwitch(leaving, entering);

    refreshViolationStatus(entering);
}
```

`replaceRow` 必须先把旧 row 的所有 coeff 从 column index 中删掉，再插入新 entries。

```cpp
void SparseTableau::replaceRow(
    int r,
    int basicVar,
    const Rat& rhs,
    std::vector<std::pair<int, Rat>> entries
) {
    // Remove old row entries from columns.
    while (!rows[r].entries.empty()) {
        int c = rows[r].entries.back().col;
        eraseCoeff(r, c);
    }

    rows[r].basicVar = basicVar;
    rows[r].rhs = rhs;
    rows[r].version++;

    if (rows[r].hasIndex) {
        rows[r].pos.clear();
    }

    for (auto [c, a] : entries) {
        if (a != 0) {
            setCoeff(r, c, a);
        }
    }
}
```

# nonBasicVars_ 的 O(1) 维护

不要每次 pivot 后在线性 vector 里找变量。维护位置数组。

```cpp
void GeneralSimplex::markBasicSwitch(int leaving, int entering) {
    // entering was non-basic, remove it from nonBasicVars_
    int pos = nonBasicPos_[entering];
    assert(pos >= 0);

    int last = nonBasicVars_.back();
    nonBasicVars_[pos] = last;
    nonBasicPos_[last] = pos;
    nonBasicVars_.pop_back();
    nonBasicPos_[entering] = -1;

    // leaving becomes non-basic
    nonBasicPos_[leaving] = static_cast<int>(nonBasicVars_.size());
    nonBasicVars_.push_back(leaving);
}
```

# recomputeBeta

`resetActiveBounds()` 或 `pop()` 后，beta dirty。重算时只给 non-basic 变量选值，然后算 basic。

```cpp
void GeneralSimplex::recomputeBeta() {
    for (int x : nonBasicVars_) {
        vars_[x].beta = chooseValueWithinBounds(x);
    }

    for (int r = 0; r < static_cast<int>(tab_.rows.size()); ++r) {
        const SparseRow& row = tab_.rows[r];
        int xb = row.basicVar;

        DeltaRat v(row.rhs);

        for (const RowEntry& e : row.entries) {
            v += DeltaRat(e.coeff) * vars_[e.col].beta;
        }

        vars_[xb].beta = v;
    }

    betaDirty_ = false;
    rebuildViolationQueue();
}
```

`chooseValueWithinBounds`：

```cpp
DeltaRat GeneralSimplex::chooseValueWithinBounds(int x) const {
    const BoundValue& l = vars_[x].lower.bound;
    const BoundValue& u = vars_[x].upper.bound;

    DeltaRat zero(0);

    if ((!l.isFinite() || zero >= l.value) &&
        (!u.isFinite() || zero <= u.value)) {
        return zero;
    }

    if (l.isFinite() && u.isFinite()) {
        return (l.value + u.value) / 2;
    }

    if (l.isFinite()) return l.value;
    if (u.isFinite()) return u.value;

    return zero;
}
```

# assertLower / assertUpper

```cpp
AssertResult GeneralSimplex::assertUpper(int x, const BoundInfo& info) {
    assert(info.bound.isFinite());
    assert(info.reason.has_value());

    BoundInfo& cur = vars_[x].upper;

    if (cur.bound.isFinite() && info.bound.value >= cur.bound.value) {
        return AssertResult::Ok;
    }

    if (vars_[x].lower.bound.isFinite() &&
        info.bound.value < vars_[x].lower.bound.value) {
        lastConflict_ = makeBoundConflict(x, /*upper=*/info, vars_[x].lower);
        return AssertResult::Conflict;
    }

    trail_.back().push_back(BoundEntry{
        .var = x,
        .isLower = false,
        .oldBound = cur
    });

    cur = info;

    if (vars_[x].basicRow == -1 && vars_[x].beta > info.bound.value) {
        updateNonBasic(x, info.bound.value);
    } else {
        refreshViolationStatus(x);
    }

    return AssertResult::Ok;
}
```

# check 主循环

```cpp
GeneralSimplexResult GeneralSimplex::check() {
    if (betaDirty_) {
        recomputeBeta();
    }

    while (true) {
        int xi = pickViolatedBasic();

        if (xi == -1) {
            return GeneralSimplexResult::Sat;
        }

        if (violatesLower(xi)) {
            int xj = findEnteringToIncrease(xi);

            if (xj == -1) {
                lastConflict_ = explainLowerConflict(xi);
                return GeneralSimplexResult::Unsat;
            }

            pivotAndUpdate(xi, xj, vars_[xi].lower.bound.value);
            continue;
        }

        if (violatesUpper(xi)) {
            int xj = findEnteringToDecrease(xi);

            if (xj == -1) {
                lastConflict_ = explainUpperConflict(xi);
                return GeneralSimplexResult::Unsat;
            }

            pivotAndUpdate(xi, xj, vars_[xi].upper.bound.value);
            continue;
        }
    }
}
```

# conflict explanation

lower conflict：

```text
xi = rhs + Σ a_j x_j
beta[xi] < lower[xi]
且没有 entering variable 可以让 xi 增大
```

说明：

```text
a_j > 0 的非基本变量都已经 at upper
a_j < 0 的非基本变量都已经 at lower
```

实现：

```cpp
TheoryConflict GeneralSimplex::explainLowerConflict(int xi) {
    int row = vars_[xi].basicRow;
    assert(row != -1);

    TheoryConflict c;

    assert(vars_[xi].lower.reason.has_value());
    c.lits.push_back(negate(*vars_[xi].lower.reason));

    for (const RowEntry& e : tab_.rows[row].entries) {
        int xj = e.col;
        const Rat& a = e.coeff;

        if (a > 0) {
            assert(atUpper(xj));
            assert(vars_[xj].upper.reason.has_value());
            c.lits.push_back(negate(*vars_[xj].upper.reason));
        } else if (a < 0) {
            assert(atLower(xj));
            assert(vars_[xj].lower.reason.has_value());
            c.lits.push_back(negate(*vars_[xj].lower.reason));
        }
    }

    return c;
}
```

upper conflict 对称：

```text
xi 太大，需要减小 xi。
a_j < 0 时，增大 x_j 可减小 xi，所以若不能动，x_j at upper。
a_j > 0 时，减小 x_j 可减小 xi，所以若不能动，x_j at lower。
```

```cpp
TheoryConflict GeneralSimplex::explainUpperConflict(int xi) {
    int row = vars_[xi].basicRow;
    assert(row != -1);

    TheoryConflict c;

    assert(vars_[xi].upper.reason.has_value());
    c.lits.push_back(negate(*vars_[xi].upper.reason));

    for (const RowEntry& e : tab_.rows[row].entries) {
        int xj = e.col;
        const Rat& a = e.coeff;

        if (a < 0) {
            assert(atUpper(xj));
            assert(vars_[xj].upper.reason.has_value());
            c.lits.push_back(negate(*vars_[xj].upper.reason));
        } else if (a > 0) {
            assert(atLower(xj));
            assert(vars_[xj].lower.reason.has_value());
            c.lits.push_back(negate(*vars_[xj].lower.reason));
        }
    }

    return c;
}
```

# 稀疏 pivot 的 fill-in 控制

Sparse tableau 最大风险是 pivot 造成 fill-in。你需要加三类控制。

## 1. entering variable 选择加入 sparsity score

Bland's rule 保证终止，但可能造成严重 fill-in。最终设计可以做：

```text
先找 eligible variables；
在 eligible 中选 fill-in cost 最小；
tie-break 用 Bland。
```

估算 fill-in cost：

```cpp
int GeneralSimplex::estimateFillIn(int row, int entering) const {
    int colNnz = tab_.cols[entering].entries.size();
    int rowNnz = tab_.rows[row].entries.size();

    return colNnz * rowNnz;
}
```

选择：

```cpp
int GeneralSimplex::chooseEnteringWithSparsity(
    int xi,
    Direction dir
) {
    int row = vars_[xi].basicRow;

    int best = -1;
    int bestCost = INT_MAX;

    for (const RowEntry& e : tab_.rows[row].entries) {
        int xj = e.col;

        if (!eligible(xi, xj, dir)) continue;

        int cost = estimateFillIn(row, xj);

        if (cost < bestCost ||
            (cost == bestCost && xj < best)) {
            best = xj;
            bestCost = cost;
        }
    }

    return best;
}
```

严格终止可以保留 Bland fallback：

```text
如果使用非 Bland 策略出现 pivot 数过多，切换 Bland。
```

## 2. drop exact zero

所有 add/update 后必须：

```cpp
if (coeff == 0) erase;
```

不要保留 exact zero。

## 3. row density 自适应

如果某行太密：

```text
nnz(row) > denseThreshold
```

可以让 row 开启 `pos` 索引，避免大量线性查找。

```cpp
void SparseTableau::afterRowMutation(int r) {
    maybeBuildRowIndex(rows[r]);
}
```

我不建议第一版就做 dense row fallback。因为 `mpq_class` dense row 内存太重。优先做：

```text
sparse vector + optional row index
```

这通常比 dense 更稳。

# rational 优化

`mpq_class` 是性能瓶颈。最终设计应该把 rational 封装起来，而不是全代码直接写 `mpq_class`。

```cpp
class Rational {
public:
    enum class Kind {
        Small, // int64 numerator / int64 denominator
        Big    // mpq_class
    };

private:
    Kind kind_;
    int64_t num_;
    int64_t den_;
    std::shared_ptr<mpq_class> big_;

public:
    Rational();
    Rational(int64_t v);
    Rational(const mpq_class& q);

    bool isZero() const;
    int sign() const;

    friend Rational operator+(const Rational&, const Rational&);
    friend Rational operator-(const Rational&, const Rational&);
    friend Rational operator*(const Rational&, const Rational&);
    friend Rational operator/(const Rational&, const Rational&);
};
```

如果现在不想实现 small rational，也至少做一层 typedef：

```cpp
using Rat = mpq_class;
```

不要让 `mpq_class` 散落在所有接口里。以后替换会容易很多。

另外：

```text
1. matrix coefficient 用 Rat；
2. rhs 用 Rat；
3. beta 用 DeltaRat；
4. bounds 用 DeltaRat；
```

不要把所有东西都用 `DeltaRat`。tableau 系数本身不含 δ，只有 assignment / bounds 需要 δ。

# 内存管理

pivot 会频繁创建临时 entries。不要每次大量小对象 malloc。

最终可用：



# 最终复杂度

dense 版：

```text
update(xj):      O(m)
find entering:   O(n)
pivot:           O(m*n)
memory:          O(m*n)
```

稀疏双索引版：

```text
update(xj):      O(nnz(column xj))
find entering:   O(nnz(row xi))
pivot:           O(nnz(column entering) * nnz(pivot row) + actual fill-in)
memory:          O(nnz(tableau))
```

这就是必须换掉 dense matrix 的根本原因。

# 你现在这段应该直接替换成什么

现在：

```cpp
// Tableau state (persistent)
// Row r:  basicVars_[r] = rhs_[r] + sum_j matrix_[r][j] * x_j
// matrix_[r][j] is meaningful for non-basic vars only.
std::vector<std::vector<mpq_class>> matrix_;  // m x n
std::vector<mpq_class> rhs_;                  // m
std::vector<int> basicVars_;                  // m
std::vector<bool> isBasic_;                   // n
int numRows_ = 0;
int numCols_ = 0;
```

替换为：

```cpp
// -------------------------------------------------------------------------
// Sparse indexed tableau state (persistent)
// -------------------------------------------------------------------------
// Row r:
//     x_b = rhs + Σ a_j * x_j
//
// Invariant:
//   - x_b = rows_[r].basicVar is basic.
//   - every x_j in rows_[r].entries is non-basic.
//   - row and column indices are cross-linked.
//   - zero coefficients are never stored.
// -------------------------------------------------------------------------

struct RowEntry {
    int col;
    mpq_class coeff;
    int colPos;
};

struct ColEntry {
    int row;
    int rowPos;
};

struct SparseRow {
    int basicVar = -1;
    mpq_class rhs = 0;
    std::vector<RowEntry> entries;

    // Optional index for large rows.
    bool hasIndex = false;
    std::unordered_map<int, int> pos;

    int version = 0;
};

struct SparseCol {
    std::vector<ColEntry> entries;
};

std::vector<SparseRow> rows_;
std::vector<SparseCol> cols_;

std::vector<VarState> vars_;

// row -> basic var
std::vector<int> basicVars_;

// non-basic variable set with O(1) update
std::vector<int> nonBasicVars_;
std::vector<int> nonBasicPos_;

// no std::vector<bool> isBasic_; use vars_[v].basicRow == -1
```

如果你不想分离 `SparseTableau` 类，也可以先把 `rows_ / cols_ / setCoeff / eraseCoeff` 放在 `GeneralSimplex` 里。但最终更干净的是单独 `SparseTableau`。

# 最重要的实现约束

这个稀疏设计里，有几个 invariant 必须一直 assert：

```cpp
void GeneralSimplex::checkInvariants() const {
#ifndef NDEBUG
    for (int r = 0; r < rows_.size(); ++r) {
        int b = rows_[r].basicVar;
        assert(vars_[b].basicRow == r);

        for (int i = 0; i < rows_[r].entries.size(); ++i) {
            const auto& e = rows_[r].entries[i];

            assert(e.coeff != 0);
            assert(vars_[e.col].basicRow == -1);

            const auto& ce = cols_[e.col].entries[e.colPos];
            assert(ce.row == r);
            assert(ce.rowPos == i);
        }
    }

    for (int c = 0; c < cols_.size(); ++c) {
        for (int k = 0; k < cols_[c].entries.size(); ++k) {
            const auto& ce = cols_[c].entries[k];
            const auto& re = rows_[ce.row].entries[ce.rowPos];

            assert(re.col == c);
            assert(re.colPos == k);
        }
    }
#endif
}
```

调 pivot 时一定要开这个。否则 sparse cross-index 出错会很难查。

# 最终建议

你的 GeneralSimplex 最终不应该是 dense tableau，也不应该是简单 `unordered_map` sparse row。最好的工程结构是：

```text
Sparse Indexed Explicit Tableau

row:
  vector<RowEntry>
  optional col->position index for large rows

column:
  vector<ColEntry>

variable:
  basicRow / bounds / beta / reason

algorithm:
  update uses column index
  entering selection scans one row
  pivot updates only rows in entering column
  dynamic constraint registration rewrites basic vars to non-basic vars
  violation queue avoids scanning all rows
  exact rational wrapper hides mpq_class
```

这套设计是 GeneralSimplex 在 SMT 求解器里最合适的最终形态：**保留 Dutertre-de Moura tableau 算法的解释能力和增量能力，同时把 dense matrix 的时间和内存浪费降到最低。**
