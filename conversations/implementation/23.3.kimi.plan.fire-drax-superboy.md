## Phase 2: RollbackUnionFind

**New files:** `src/theory/euf/RollbackUnionFind.h`, `.cpp`

```cpp
class RollbackUnionFind {
public:
    EufClassId addNode();
    EufClassId find(EufClassId x) const;      // NO path compression
    bool same(EufClassId a, EufClassId b) const;
    bool unite(EufClassId a, EufClassId b);   // union-by-size; returns true iff merged
    size_t snapshot() const;
    void rollback(size_t snap);
    uint32_t classSize(EufClassId root) const;

private:
    std::vector<EufClassId> parent_;
    std::vector<uint32_t> size_;
    struct Change {
        EufClassId childRoot;
        EufClassId parentRoot;
        uint32_t oldParentSize;
    };
    std::vector<Change> trail_;
};
```

**Rules:**
- `find()`: iteratively walk `parent_` links; read-only, no compression.
- `unite(a,b)`: if `ra == rb`, return `false` immediately (NO trail push).
- If merge: attach smaller root to larger root; push `{rb, ra, oldRaSize}` to `trail_`.
- `rollback()`: pop `trail_`; restore `parent_[childRoot] = childRoot` and `size_[parentRoot] = oldParentSize`.

**Unit test:** Rollback to various snapshots; verify `same()` correctness.

### 3.4 Merge (processes to FIXPOINT)

```cpp
bool IncrementalEGraph::merge(EufTermId a, EufTermId b, const MergeReason& reason) {
    ensureFreshSignatureTable();
    mergeQueue_.push_back({a, b, reason});
    bool any = false;
    while (!mergeQueue_.empty()) {
        auto m = mergeQueue_.front();
        mergeQueue_.pop_front();

        if (same(m.a, m.b)) continue;
        any = true;

        EufClassId ra = rep(m.a);
        EufClassId rb = rep(m.b);

        // sort check
        if (tm_.node(m.a).sort != tm_.node(m.b).sort &&
            tm_.node(m.a).sort != NullSort && tm_.node(m.b).sort != NullSort) {
            continue; // sort mismatch; let solver handle as unknown
        }

        // collect affected parents BEFORE unite
        std::vector<EufTermId> affected = collectParents(ra);
        auto affectedB = collectParents(rb);
        affected.insert(affected.end(), affectedB.begin(), affectedB.end());

        // record merge BEFORE unite
        MergeRecord rec;
        rec.id = static_cast<MergeId>(mergeRecords_.size());
        rec.lhs = m.a; rec.rhs = m.b;
        rec.lhsRootBefore = ra; rec.rhsRootBefore = rb;
        rec.reason = m.reason;
        mergeRecords_.push_back(rec);

        // unite (ra is winner, rb is loser)
        uf_.unite(ra, rb);

        // merge members: append rb members to ra
        memberTrail_.push_back({ra, rb, members_[ra].size()});
        members_[ra].insert(members_[ra].end(), members_[rb].begin(), members_[rb].end());
        // DO NOT clear members_[rb]; it becomes stale but is never read unless rb becomes root again

        // enqueue affected parents
        for (EufTermId p : affected) {
            congruenceQueue_.push_back(p);
        }

        // process congruence queue
        processCongruenceQueue();
    }
    return any;
}
```

### 3.5 Congruence queue

```cpp
void IncrementalEGraph::processCongruenceQueue() {
    while (!congruenceQueue_.empty()) {
        EufTermId app = congruenceQueue_.front();
        congruenceQueue_.pop_front();

        if (tm_.node(app).args.empty()) continue;

        AppSignature sig = computeSignature(app);
        auto it = sigTable_.find(sig);

        if (it == sigTable_.end()) {
            sigTable_.emplace(sig, app);
            continue;
        }

        EufTermId existing = it->second;

        // stale check
        if (computeSignature(existing) != sig) {
            sigTable_[sig] = app; // overwrite stale
            continue;
        }

        if (!same(existing, app)) {
            MergeReason cr;
            cr.kind = MergeReasonKind::Congruence;
            cr.app1 = existing; cr.app2 = app;
            const auto& n1 = tm_.node(existing);
            const auto& n2 = tm_.node(app);
            for (size_t i = 0; i < n1.args.size(); ++i) {
                cr.argPairs.push_back({n1.args[i], n2.args[i]});
            }
            mergeQueue_.push_back({existing, app, cr});
            // mergeQueue will be consumed by outer merge() loop
        }
    }
}
```

### 3.6 Signature table lazy rebuild

```cpp
void IncrementalEGraph::ensureFreshSignatureTable() {
    if (!sigTableDirty_) return;
    sigTable_.clear();
    for (EufTermId t = 0; t < tm_.termCount(); ++t) {
        if (tm_.node(t).args.empty()) continue;
        AppSignature sig = computeSignature(t);
        auto it = sigTable_.find(sig);
        if (it == sigTable_.end()) {
            sigTable_.emplace(sig, t);
        } else if (!same(it->second, t)) {
            // congruent pair found during rebuild
            MergeReason cr;
            cr.kind = MergeReasonKind::Congruence;
            cr.app1 = it->second; cr.app2 = t;
            const auto& n1 = tm_.node(it->second);
            const auto& n2 = tm_.node(t);
            for (size_t i = 0; i < n1.args.size(); ++i) {
                cr.argPairs.push_back({n1.args[i], n2.args[i]});
            }
            mergeQueue_.push_back({it->second, t, cr});
        }
    }
    sigTableDirty_ = false;
}
```

**When called:** At start of `merge()`.

### 3.7 Rollback

```cpp
void IncrementalEGraph::rollback(EGraphSnapshot snap) {
    uf_.rollback(snap.ufTrailSize);
    mergeRecords_.resize(snap.mergeRecordSize);

    while (memberTrail_.size() > snap.memberTrailSize) {
        auto ch = memberTrail_.back();
        memberTrail_.pop_back();
        members_[ch.dstRoot].resize(ch.oldDstSize);
        // members_[ch.srcRoot] is stale; do not touch
    }

    mergeQueue_.resize(snap.mergeQueueSize);
    congruenceQueue_.clear(); // safest: rebuild will re-detect

    sigTableDirty_ = true;
}
```

### 3.8 Explanation

```cpp
ExplainResult IncrementalEGraph::explainEquality(EufTermId a, EufTermId b) {
    if (a == b) return {true, {}};
    if (!same(a, b)) return {false, {}};

    // Build adjacency from active mergeRecords
    std::unordered_map<EufTermId, std::vector<std::pair<EufTermId, MergeId>>> adj;
    for (const auto& rec : mergeRecords_) {
        adj[rec.lhs].push_back({rec.rhs, rec.id});
        adj[rec.rhs].push_back({rec.lhs, rec.id});
    }

    std::unordered_map<EufTermId, EufTermId> parentTerm;
    std::unordered_map<EufTermId, MergeId> parentMerge;
    std::queue<EufTermId> q;
    std::unordered_set<EufTermId> visited;

    q.push(a); visited.insert(a); parentTerm[a] = a;

    while (!q.empty()) {
        EufTermId cur = q.front(); q.pop();
        auto it = adj.find(cur);
        if (it == adj.end()) continue;
        for (auto [next, mid] : it->second) {
            if (visited.count(next)) continue;
            visited.insert(next);
            parentTerm[next] = cur;
            parentMerge[next] = mid;
            if (next == b) {
                std::vector<SatLit> reasons;
                std::unordered_set<uint64_t> activePairs;
                EufTermId node = b;
                while (node != a) {
                    MergeId mid = parentMerge[node];
                    auto er = explainMergeReason(mergeRecords_[mid].reason, activePairs);
                    if (!er.ok) return {false, {}};
                    appendUnique(reasons, er.reasons);
                    node = parentTerm[node];
                }
                return {true, std::move(reasons)};
            }
            q.push(next);
        }
    }
    return {false, {}};
}
```

`explainMergeReason` with cycle guard:
```cpp
ExplainResult explainMergeReason(const MergeReason& r,
                                 std::unordered_set<uint64_t>& activePairs) {
    if (r.kind == MergeReasonKind::AssertedEquality) {
        return {true, {r.assertedLit}};
    }
    std::vector<SatLit> out;
    for (auto [ai, bi] : r.argPairs) {
        if (ai == bi) continue;
        uint64_t pk = (static_cast<uint64_t>(ai) << 32) | bi;
        if (ai > bi) pk = (static_cast<uint64_t>(bi) << 32) | ai;
        if (!activePairs.insert(pk).second) return {false, {}};
        auto sub = explainEquality(ai, bi, activePairs);
        activePairs.erase(pk);
        if (!sub.ok) return {false, {}};
        appendUnique(out, sub.reasons);
    }
    return {true, std::move(out)};
}
```

### 4.4 pop()

```cpp
void EufSolver::pop(uint32_t n) {
    while (n-- && !scopeLimits_.empty()) {
        size_t limit = scopeLimits_.back();
        EGraphSnapshot snap = scopeSnapshots_.back();

        trail_.resize(limit);

        // remove disequalities whose assignment was popped
        auto dit = std::remove_if(disequalities_.begin(), disequalities_.end(),
            [limit, this](const auto& d) {
                // find assignment index; if index >= limit, remove
                // simpler: store index in disequality, or use level
                return d.level > (trail_.empty() ? 0 : trail_.back().level);
            });
        disequalities_.erase(dit, disequalities_.end());

        egraph_.rollback(snap);
        pendingConflict_.reset();

        scopeLimits_.pop_back();
        scopeSnapshots_.pop_back();
    }
}
```

**Simpler disequality cleanup:** Store `size_t trailIndex` in `ActiveDisequality`. On pop to limit, remove all with `trailIndex >= limit`.

### 4.5 backtrackToLevel()

```cpp
void EufSolver::backtrackToLevel(int level) {
    // remove trail entries above level
    auto it = std::remove_if(trail_.begin(), trail_.end(),
        [level](const auto& a){ return a.level > level; });
    trail_.erase(it, trail_.end());

    // rollback to nearest scope snapshot
    while (!scopeLimits_.empty() && scopeLimits_.back() > trail_.size()) {
        scopeLimits_.pop_back();
        scopeSnapshots_.pop_back();
    }

    if (!scopeSnapshots_.empty()) {
        egraph_.rollback(scopeSnapshots_.back());
    } else {
        egraph_.rollback(EGraphSnapshot{});
    }

    // remove disequalities above level
    auto dit = std::remove_if(disequalities_.begin(), disequalities_.end(),
        [level](const auto& d){ return d.level > level; });
    disequalities_.erase(dit, disequalities_.end());

    pendingConflict_.reset();
}
```

### 4.6 assertLit() — INCREMENTAL, complete Bool handling

```cpp
void EufSolver::assertLit(const TheoryAtomRecord& atom, bool value, int level, SatLit reason) {
    if (!std::holds_alternative<EufAtomPayload>(atom.payload)) return;

    const auto& payload = std::get<EufAtomPayload>(atom.payload);

    // Intern terms (monotonic, no rollback)
    EufTermId lhs = termManager_.intern(payload.lhs, *coreIr_);
    EufTermId rhs = termManager_.intern(payload.rhs, *coreIr_);
    if (lhs == NullEufTerm || rhs == NullEufTerm) return;

    egraph_.ensureTerm(lhs);
    egraph_.ensureTerm(rhs);

    // Bool formula-position semantics:
    // Atomizer encodes Bool terms as (= term true)
    // If (= b true) assigned true  -> merge(b, true)
    // If (= b true) assigned false -> merge(b, false)
    if (payload.rel == Relation::Eq && payload.rhs == TrueSentinelExpr) {
        EufTermId target = value ? termManager_.trueConstant() : termManager_.falseConstant();
        if (target != NullEufTerm) {
            trail_.push_back({level, reason, atom, value});
            egraph_.merge(lhs, target, MergeReason::asserted(reason));
            return;
        }
    }

    // Effective relation for normal EUF equality/disequality
    bool isEq = false;
    if (payload.rel == Relation::Eq) {
        isEq = value;
    } else if (payload.rel == Relation::Neq) {
        isEq = !value;
    } else {
        return; // unsupported
    }

    trail_.push_back({level, reason, atom, value});

    if (isEq) {
        egraph_.merge(lhs, rhs, MergeReason::asserted(reason));
    } else {
        disequalities_.push_back({lhs, rhs, reason, level});
    }
}
```

### 4.7 check()

```cpp
TheoryCheckResult EufSolver::check(TheoryLemmaDatabase& /*lemmaDb*/) {
    // 1. pending conflict from assertLit
    if (pendingConflict_) {
        TheoryConflict c = *pendingConflict_;
        pendingConflict_.reset();
        return TheoryCheckResult::mkConflict(std::move(c));
    }

    // 2. true/false invariant (defense in depth)
    EufTermId tTrue = termManager_.trueConstant();
    EufTermId tFalse = termManager_.falseConstant();
    if (tTrue != NullEufTerm && tFalse != NullEufTerm && egraph_.same(tTrue, tFalse)) {
        auto er = egraph_.explainEquality(tTrue, tFalse);
        if (er.ok) {
            for (auto& lit : er.reasons) lit = lit.negated();
            return TheoryCheckResult::mkConflict(TheoryConflict{std::move(er.reasons)});
        }
        return TheoryCheckResult::unknown();
    }

    // 3. disequality conflicts
    for (const auto& d : disequalities_) {
        if (egraph_.same(d.lhs, d.rhs)) {
            auto er = egraph_.explainEquality(d.lhs, d.rhs);
            if (er.ok) {
                for (auto& lit : er.reasons) lit = lit.negated();
                er.reasons.push_back(d.reason.negated());
                return TheoryCheckResult::mkConflict(TheoryConflict{std::move(er.reasons)});
            }
            return TheoryCheckResult::unknown();
        }
    }

    return TheoryCheckResult::consistent();
}
```

### 4.8 PendingConflict handling in assertLit

If `egraph_.merge()` causes a disequality to be violated during `assertLit()`, set `pendingConflict_`:

```cpp
// Inside assertLit, after merge:
if (!isEq && egraph_.same(lhs, rhs)) {
    auto er = egraph_.explainEquality(lhs, rhs);
    if (er.ok) {
        for (auto& lit : er.reasons) lit = lit.negated();
        er.reasons.push_back(reason.negated());
        pendingConflict_ = TheoryConflict{std::move(er.reasons)};
    }
}
```

`check()` consumes and clears `pendingConflict_`. `backtrackToLevel()` and `pop()` clear it unconditionally.

## Hard Rules (soundness-critical)

1. **Bool formula-position:** `(= b true)` assigned false → `merge(b, false)`.
2. **check() order:** (1) pendingConflict, (2) true/false conflict, (3) disequality conflicts, (4) consistent.
3. **pendingConflict_ cleared on backtrack/pop:** unconditionally reset.
4. **push/pop use scopeSnapshots_:** `EGraphSnapshot` stored at push, restored at pop.
5. **explainEquality returns ExplainResult:** caller must check `er.ok`; if false, return `Unknown`.
6. **Rollback clears queues:** `mergeQueue_` resized, `congruenceQueue_` cleared, `sigTableDirty_ = true`.
7. **members_[nonRoot] may be stale:** only read `members_[root]` where root is current UF root.
8. **RollbackUnionFind no-op:** `ra == rb` → return false, do NOT push to trail.
9. **merge() processes to fixpoint:** returns only after `mergeQueue` and `congruenceQueue` empty.
10. **Stale signatures detected:** `computeSignature(storedTerm) != sig` → overwrite stale entry.
11. **Model construction and propagation are strictly out of scope:** do not implement.
