## What This Plan Actually Changes

The ONLY changes are internal to `IncrementalEGraph` performance:

1. **Replace** `sigTable_` (plain unordered_map + `sigTableDirty_` flag + full rebuild) with `RollbackSignatureTable` (trail-based rollback, no rebuild).
2. **Add** `currentSig_` per-app signature tracking with rollback trail.
3. **Replace** `explainEquality()` (full-scan `mergeRecords_` + BFS) with `ProofForest` (incremental adjacency + tree-path BFS).
4. **Add** `ExplainContext` local memo for recursive congruence explanation.

No changes to `EufSolver` public behavior, `EufTypes` semantics, `RollbackUnionFind`, or `EufTermManager`.

## Detailed Design

### 1. RollbackSignatureTable

```cpp
class RollbackSignatureTable {
public:
    std::optional<EufTermId> find(const AppSignature& sig) const;
    void insertOrAssign(const AppSignature& sig, EufTermId owner);
    void eraseIfOwner(const AppSignature& sig, EufTermId owner); // owner-check mandatory
    size_t snapshot() const;
    void rollback(size_t snap);
    void clear();
private:
    enum class ChangeKind { InsertNew, Overwrite, Erase };
    struct Change { ChangeKind kind; AppSignature sig; std::optional<EufTermId> oldOwner; };
    std::unordered_map<AppSignature, EufTermId, AppSignatureHash> table_;
    std::vector<Change> trail_;
};
```

Key invariant: `eraseIfOwner` refuses to erase if the stored owner does not match. This prevents one app's refresh from deleting another valid app's signature entry.

### 2. currentSig rollback

```cpp
std::vector<std::optional<AppSignature>> currentSig_;   // per-app registered signature
std::vector<std::pair<EufTermId, std::optional<AppSignature>>> currentSigTrail_;

void setCurrentSig(EufTermId app, std::optional<AppSignature> sig) {
    if (currentSig_[app] == sig) return;
    currentSigTrail_.push_back({app, currentSig_[app]});
    currentSig_[app] = sig;
}
```

Rollback: pop trail, restore `currentSig_` values.

### 3. refreshSignature (replaces ensureFreshSignatureTable)

```cpp
void refreshSignature(EufTermId app) {
    // 1. unregister old
    if (currentSig_[app]) {
        sigTable_.eraseIfOwner(*currentSig_[app], app);
        setCurrentSig(app, std::nullopt);
    }
    // 2. compute new
    AppSignature sig = computeSignature(app);
    // 3. find existing owner
    auto ownerOpt = sigTable_.find(sig);
    if (!ownerOpt) {
        sigTable_.insertOrAssign(sig, app);
        setCurrentSig(app, sig);
        return;
    }
    EufTermId owner = *ownerOpt;
    // 4. stale owner check
    if (!currentSig_[owner] || *currentSig_[owner] != sig) {
        sigTable_.insertOrAssign(sig, app);
        setCurrentSig(app, sig);
        return;
    }
    // 5. congruence merge needed?
    if (!same(owner, app)) {
        enqueueCongruenceMerge(owner, app);
        // DO NOT register app; merge will change reps, then re-process
        return;
    }
    // 6. same class
    sigTable_.insertOrAssign(sig, owner);
    setCurrentSig(app, sig);
}
```

### 4. ProofForest

```cpp
class ProofForest {
public:
    void addEdge(EufTermId u, EufTermId v, const MergeReason& reason);
    size_t snapshot() const { return activeEdgeCount_; }
    void rollback(size_t snap) { activeEdgeCount_ = snap; }
    void clear();
    std::vector<size_t> path(EufTermId u, EufTermId v) const;
private:
    struct Edge { EufTermId u; EufTermId v; MergeReason reason; };
    std::vector<Edge> edges_;
    std::vector<std::vector<std::pair<EufTermId, size_t>>> adj_;
    size_t activeEdgeCount_ = 0;
};
```

`path()` does BFS over adjacency, skipping `edgeId >= activeEdgeCount_`. Since edges are only added when `!same(u,v)`, the active subgraph is a forest.

### 5. Explanation rewrite

```cpp
ExplainResult explainEquality(EufTermId a, EufTermId b) {
    if (a == b) return {true, {}};
    if (!same(a, b)) return {false, {}};

    auto edgeIds = proofForest_.path(a, b);
    if (edgeIds.empty() && a != b) return {false, {}};

    ExplainContext ctx;
    std::vector<SatLit> reasons;
    for (size_t eid : edgeIds) {
        auto sub = explainEdge(eid, ctx);
        if (!sub.ok) return {false, {}};
        appendDedup(reasons, sub.reasons);
    }
    return {true, std::move(reasons)};
}
```

`explainEdge` handles asserted equality (returns the literal) or congruence (recursively explains arg pairs via `explainEquality` with `ExplainContext` memo).

### 7. Rollback order

```cpp
void rollback(EGraphSnapshot s) {
    mergeQueue_.resize(s.mergeQueueSize);
    congruenceQueue_.clear();

    proofForest_.rollback(s.proofForestSnap);
    mergeRecords_.resize(s.mergeRecordSize);

    rollbackCurrentSig(s.currentSigSnap);
    sigTable_.rollback(s.sigTableSnap);

    rollbackMembers(s.memberTrailSize);
    uf_.rollback(s.ufTrailSize);
}
```
