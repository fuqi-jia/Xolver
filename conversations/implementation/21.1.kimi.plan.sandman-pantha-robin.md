# EUF Solver Implementation Plan for NLColver

## Goal
Implement an independent EUF (Equality with Uninterpreted Functions) theory solver for NLColver, supporting QF_UF and laying groundwork for QF_UFLIA/UFLRA/UFNIA/UFNRA.

## Architecture Context

NLColver uses:
- **CoreIr** as lightweight dense array IR (`CoreExpr` nodes with `Kind`, `SortId`, `children`, `Payload`)
- **SOMTParser** for parsing; `FrontendAdapter` maps SOMTParser `Node` → `CoreExpr`
- **Atomizer** converts `CoreExpr` assertions to SAT clauses + `TheoryAtomRecord`s
- **TheoryAtomRegistry** deduplicates theory atoms (`LinearAtomPayload`, `PolynomialAtomPayload` variants)
- **TheoryManager** dispatches `assertLit`/`check` to registered `TheorySolver`s by `TheoryId`
- **Solver::Impl** registers solvers based on `(set-logic ...)` string
- V1 theory solvers use **full-rebuild** `check()`: `assertLit` stores active assignments, `check()` rebuilds from scratch

## Key Insight: Current UF Gap

`FrontendAdapter::mapKind` maps `NT_UF_APPLY` → `Kind::Unknown`. This means all uninterpreted function applications become `Kind::Unknown` in CoreIr, making them invisible to the atomizer and theory solvers.

### 2.3 `EufTermManager`

Interns EUF terms from CoreExpr subgraphs. Two-level interning:
1. **Function symbols**: name string → `FuncSymbolId`
2. **Terms**: `(FuncSymbolId, vector<EufTermId>)` → `EufTermId`

Constants = 0-ary functions.

```cpp
class EufTermManager {
public:
    struct ENode {
        EufTermId id;
        FuncSymbolId symbol;
        std::vector<EufTermId> args;  // empty for constants
        SortId sort;
        ExprId origin;  // corresponding CoreExpr id
    };

    // Intern a CoreExpr subtree. Returns NullEufTerm if non-UF term encountered.
    EufTermId intern(ExprId exprId, const CoreIr& ir);

    const ENode& node(EufTermId id) const;
    bool isApplication(EufTermId id) const;
    size_t termCount() const;

    void clear();

private:
    struct TermKey {
        FuncSymbolId symbol;
        std::vector<EufTermId> args;
        bool operator==(const TermKey&) const = default;
    };
    struct TermKeyHash {
        std::size_t operator()(const TermKey& k) const;
    };

    std::vector<ENode> nodes_;
    std::unordered_map<std::string, FuncSymbolId> symbolMap_;
    std::unordered_map<TermKey, EufTermId, TermKeyHash> termMap_;

    FuncSymbolId internSymbol(const std::string& name);
    EufTermId createNode(FuncSymbolId sym, std::vector<EufTermId> args, SortId sort, ExprId origin);
};
```

**Interning logic** (`internRec`):
- `Kind::Variable` with uninterpreted sort → constant (0-ary function, symbol = var name)
- `Kind::UFApply` → intern symbol from payload name, intern each child recursively, then intern application
- Arithmetic `Kind::Add/Sub/Mul/...` → return `NullEufTerm` (not handled by EUF)
- Nested `Kind::Eq/Distinct` inside arguments → return `NullEufTerm` (EUf only handles equality at top level)

### 2.4 `EGraph` (Full Rebuild)

V1 uses full rebuild: `check()` calls `egraph_.clear()` then rebuilds from active assignments.

```cpp
class EGraph {
public:
    void clear();

    // Add a term node (must be called for all terms before any merge)
    void addTerm(EufTermId t);

    // Merge two terms' equivalence classes. Records merge reason for explanation.
    void merge(EufTermId a, EufTermId b, const MergeReason& reason);

    bool same(EufTermId a, EufTermId b) const;
    EClassId find(EufTermId t) const;

    // Close congruence: repeatedly scan all applications, merge congruent pairs.
    // V1 uses simple repeated-scan algorithm (correctness over speed).
    void closeCongruence(const EufTermManager& tm);

    // Get the representative class of a term
    EClassId getClass(EufTermId t) const;

    // Get all members of a class
    const std::vector<EufTermId>& classMembers(EClassId c) const;

    // Build model: assign a unique value per class
    std::unordered_map<EClassId, std::string> buildModel() const;

    // Explain why two terms are in the same class.
    // Returns the set of asserted equality literals that justify the merge.
    std::vector<SatLit> explainEquality(EufTermId a, EufTermId b,
                                        const EufTermManager& tm) const;

private:
    struct EClass {
        EClassId id;
        EClassId parent;
        std::vector<EufTermId> members;
    };

    struct ExplainEdge {
        EClassId from;
        EClassId to;
        MergeReason reason;
    };

    std::vector<EClass> classes_;
    std::vector<ExplainEdge> explainEdges_;

    // Union-Find with path compression
    EClassId ufFind(EClassId x) const;
    void ufUnite(EClassId a, EClassId b);

    // Build canonical app key for congruence detection
    struct AppKey {
        FuncSymbolId symbol;
        std::vector<EClassId> argClasses;
        bool operator==(const AppKey&) const = default;
    };
    struct AppKeyHash {
        std::size_t operator()(const AppKey& k) const;
    };

    // Rebuild all application keys and detect congruent pairs
    std::vector<std::pair<EufTermId, EufTermId>> findCongruentPairs(
        const EufTermManager& tm) const;
};
```

**Congruence closure algorithm (V1 simple repeated-scan):**
```cpp
void EGraph::closeCongruence(const EufTermManager& tm) {
    bool changed = true;
    while (changed) {
        changed = false;
        auto pairs = findCongruentPairs(tm);
        for (auto [a, b] : pairs) {
            if (!same(a, b)) {
                MergeReason reason;
                reason.kind = MergeReasonKind::Congruence;
                reason.lhsApp = a;
                reason.rhsApp = b;
                // Record arg equalities for explanation
                const auto& na = tm.node(a);
                const auto& nb = tm.node(b);
                for (size_t i = 0; i < na.args.size(); ++i) {
                    reason.argPairs.push_back({na.args[i], nb.args[i]});
                }
                merge(a, b, reason);
                changed = true;
            }
        }
    }
}
```

**Explanation algorithm:**
- Build temporary undirected graph from `explainEdges_`
- BFS from `class(a)` to `class(b)` to find path
- For each edge on path:
  - If `AssertedEquality` → add `literal` to reasons
  - If `Congruence` → recursively explain each `argPairs[i]` equality
- Collect unique literals

Then `check()`:
```cpp
TheoryCheckResult EufSolver::check(TheoryLemmaDatabase&) {
    termManager_.clear();
    egraph_.clear();
    equalities_.clear();
    disequalities_.clear();

    // 1. Build effective atoms and intern terms
    for (const auto& a : activeAssignments_) {
        Relation effRel = a.value ? a.rawRel
                        : (a.rawRel == Relation::Eq ? Relation::Neq : Relation::Eq);

        EufTermId lhs = termManager_.intern(a.lhsExpr, coreIr_);
        EufTermId rhs = termManager_.intern(a.rhsExpr, coreIr_);

        if (lhs == NullEufTerm || rhs == NullEufTerm) {
            // Contains non-UF subterms (e.g., arithmetic inside UF args)
            // V1: mark unsupported
            return TheoryCheckResult::unknown();
        }

        if (effRel == Relation::Eq) {
            equalities_.push_back({lhs, rhs, effRel, a.lit});
        } else {
            disequalities_.push_back({lhs, rhs, effRel, a.lit});
        }
    }

    // 2. Register all terms in EGraph
    for (EufTermId t = 0; t < termManager_.termCount(); ++t) {
        egraph_.addTerm(t);
    }

    // 3. Assert equalities
    for (const auto& eq : equalities_) {
        MergeReason reason;
        reason.kind = MergeReasonKind::AssertedEquality;
        reason.literal = eq.reason;
        egraph_.merge(eq.lhs, eq.rhs, reason);
    }

    // 4. Close congruence
    egraph_.closeCongruence(termManager_);

    // 5. Check disequalities
    for (const auto& neq : disequalities_) {
        if (egraph_.same(neq.lhs, neq.rhs)) {
            auto eqReasons = egraph_.explainEquality(neq.lhs, neq.rhs, termManager_);
            // Conflict clause: negation of all equality reasons + negation of disequality
            std::vector<SatLit> clause;
            for (auto lit : eqReasons) {
                clause.push_back(lit.negated());
            }
            clause.push_back(neq.reason.negated());
            return TheoryCheckResult::mkConflict(TheoryConflict{clause});
        }
    }

    // 6. Optional propagation (V1 can skip)
    // if (auto lemma = findPropagation()) return TheoryCheckResult::mkLemma(*lemma);

    return TheoryCheckResult::consistent();
}
```

**Note:** `EufSolver` needs access to `CoreIr` for interning. Add `setCoreIr(const CoreIr* ir)` or store it during setup.

### 4.4 `backtrackToLevel`

Remove assignments with `level > target`:
```cpp
void EufSolver::backtrackToLevel(int level) {
    auto it = std::remove_if(activeAssignments_.begin(), activeAssignments_.end(),
        [level](const auto& a) { return a.level > level; });
    activeAssignments_.erase(it, activeAssignments_.end());
}
```

### 4.5 `push`/`pop`

```cpp
void EufSolver::push() {
    scopeStack_.push_back(activeAssignments_);
}
void EufSolver::pop(uint32_t n) {
    for (uint32_t i = 0; i < n && !scopeStack_.empty(); ++i) {
        activeAssignments_ = std::move(scopeStack_.back());
        scopeStack_.pop_back();
    }
}
```

## Red Lines

1. **Different sorts never merge.** `EGraph::merge` must check `termManager_.node(a).sort == termManager_.node(b).sort`.
2. **Congruence explanation must be recursive.** `explainEquality` on a congruence merge must recursively explain each argument pair equality, ultimately reaching only asserted equality literals.
3. **Conflict clause must include disequality reason.** A violated `distinct(a,b)` must produce `¬(eq reasons) ∨ ¬(distinct lit)`.
4. **V1 uses full rebuild only.** No incremental e-graph, no rollbackable union-find. Correctness first.
5. **Mixed arithmetic/UF terms return Unknown in V1.** `f(x+1)` where `x` is Int contains arithmetic inside UF args; intern returns `NullEufTerm`, solver returns `Unknown`.
6. **No Nelson-Oppen in V1.** EUF and arithmetic solvers run independently; shared equalities are not propagated between them.
