# EUF Solver Implementation Plan for NLColver — Revised V2

## Goal
Implement a **sound** QF_UF (equality-only fragment with uninterpreted functions) theory solver. V1 is strictly limited to pure QF_UF — no mixed theories, no quantifiers.

## Critical V1 Constraints (Red Lines)

1. **NO mixed logics in V1.** Do NOT register QF_UFLIA/UFLRA/UFNIA/UFNRA. Without Nelson-Oppen, these are **unsound**.
2. **Unsupported logics must NOT silently fall back to LRA.** Return `Unknown` explicitly.
3. **Term-level proof graph for explanation.** Never use current class root as BFS endpoint.
4. **Bool-valued UF predicates must work.** `(p a)` where `p` returns Bool → `(= (p a) true)`.
5. **Both `true` and `false` EUF constants required.** EGraph must enforce `true != false`.
6. **N-ary equality/distinct must normalize** with unique synthetic ExprIds per pairwise atom.
7. **Function symbol key includes full signature** (name + arg sorts + result sort).
8. **Conflict clause uses `assertedLit` negation** — the actual true-assigned SAT literal.
9. **Explanation failure is an error, not empty reasons.**
10. **Sort mismatch is an internal error** — not silent ignore.
11. **Trail-based push/pop + backtrack-safe scope limits.**
12. **TheoryAtomRegistry dedup: get-or-create API.** Don't create duplicate SAT vars.

---

## Architecture Context

NLColver uses:
- **CoreIr** as lightweight dense array IR (`CoreExpr` nodes with `Kind`, `SortId`, `children`, `Payload`)
- **SOMTParser** for parsing; `FrontendAdapter` maps SOMTParser `Node` → `CoreExpr`
- **Atomizer** converts `CoreExpr` assertions to SAT clauses + `TheoryAtomRecord`s
- **TheoryAtomRegistry** deduplicates theory atoms (`LinearAtomPayload`, `PolynomialAtomPayload` variants)
- **TheoryManager** dispatches `assertLit`/`check` to registered `TheorySolver`s by `TheoryId`
- **Solver::Impl** registers solvers based on `(set-logic ...)` string
- V1 theory solvers use **full-rebuild** `check()`

### 2.2 N-ary Equality / Distinct Normalization

SMT-LIB allows n-ary `=` and `distinct`:
```smt2
(= a b c)         ; means a=b ∧ b=c
(distinct a b c)  ; means a≠b ∧ a≠c ∧ b≠c
```

**Where to normalize:** In `Atomizer::atomizeRec`, before extracting theory atoms. Only applies when `defaultTheory_ == TheoryId::EUF`.

**Implementation in Atomizer:**

Add to `Atomizer` class:
```cpp
private:
    SyntheticExprIdAllocator synthExprAlloc_;

    SatLit atomizeNaryEufEq(ExprId eid, const CoreIr& ir);
    SatLit atomizeNaryEufDistinct(ExprId eid, const CoreIr& ir);
```

```cpp
SatLit Atomizer::atomizeNaryEufEq(ExprId eid, const CoreIr& ir) {
    const auto& e = ir.get(eid);
    SatVar andVar = freshVar();
    SatLit andLit = SatLit::positive(andVar);

    std::vector<SatLit> pairwiseLits;
    for (size_t i = 0; i + 1 < e.children.size(); ++i) {
        ExprId lhs = e.children[i];
        ExprId rhs = e.children[i + 1];
        if (lhs > rhs) std::swap(lhs, rhs);

        ExprId synthExpr = synthExprAlloc_.next();
        SatVar pv = freshVar();
        SatLit plit = SatLit::positive(pv);
        pairwiseLits.push_back(plit);

        if (registry_) {
            registry_->registerParsedTheoryAtom(
                pv, synthExpr, TheoryId::EUF,
                EufAtomPayload{lhs, rhs, Relation::Eq});
        }
    }

    // Tseitin: andVar → each pairwise
    for (SatLit pl : pairwiseLits) {
        sat_.addClause({SatLit::negative(andVar), pl});
    }
    // Tseitin: any pairwise → andVar
    std::vector<SatLit> clause;
    clause.push_back(SatLit::positive(andVar));
    for (SatLit pl : pairwiseLits) {
        clause.push_back(pl.negated());
    }
    sat_.addClause(clause);

    return andLit;
}

SatLit Atomizer::atomizeNaryEufDistinct(ExprId eid, const CoreIr& ir) {
    const auto& e = ir.get(eid);
    SatVar andVar = freshVar();
    SatLit andLit = SatLit::positive(andVar);

    std::vector<SatLit> pairwiseLits;
    for (size_t i = 0; i < e.children.size(); ++i) {
        for (size_t j = i + 1; j < e.children.size(); ++j) {
            ExprId lhs = e.children[i];
            ExprId rhs = e.children[j];
            if (lhs > rhs) std::swap(lhs, rhs);

            ExprId synthExpr = synthExprAlloc_.next();
            SatVar pv = freshVar();
            SatLit plit = SatLit::positive(pv);
            pairwiseLits.push_back(plit);

            if (registry_) {
                registry_->registerParsedTheoryAtom(
                    pv, synthExpr, TheoryId::EUF,
                    EufAtomPayload{lhs, rhs, Relation::Neq});
            }
        }
    }

    // Tseitin AND encoding
    for (SatLit pl : pairwiseLits) {
        sat_.addClause({SatLit::negative(andVar), pl});
    }
    std::vector<SatLit> clause;
    clause.push_back(SatLit::positive(andVar));
    for (SatLit pl : pairwiseLits) {
        clause.push_back(pl.negated());
    }
    sat_.addClause(clause);

    return andLit;
}
```

**Critical:** Each pairwise atom gets a **unique synthetic ExprId**. This avoids exprId collision in `TheoryAtomRegistry` and diagnostics.

### 2.4 Modify `atomizeRec` default case for EUF

In the `default` case (theory atom extraction), add EUF path before existing arithmetic paths:

```cpp
if (defaultTheory_ == TheoryId::EUF) {
    if (e.kind == Kind::Eq || e.kind == Kind::Distinct) {
        if (e.children.size() == 2) {
            Relation rel = (e.kind == Kind::Eq) ? Relation::Eq : Relation::Neq;

            return result;
        } else {
            registry_->setUnsupportedTheorySeen();

    } else {
        // In QF_UF, only Eq/Distinct and Bool predicates are theory atoms
        registry_->setUnsupportedTheorySeen();
    }
```

```cpp
    };
```

```cpp
        std::size_t operator()(const TermKey& k) const;
    };
    std::unordered_map<TermKey, EufTermId, TermKeyHash> termMap_;

    EufTermId falseConstant_ = NullEufTerm;

    FuncSymbolId internSymbol(const FuncSymbolKey& key);

    EufTermId internTrueConstant();
    EufTermId internFalseConstant();

```

**Interning logic** (`internRec`):
- `TrueSentinelExpr` → `trueConstant()`
- `FalseSentinelExpr` → `falseConstant()`
- `Kind::Variable` with any sort → constant (0-ary function, symbol = var name)
- `Kind::ConstBool` with value `true` → `trueConstant()`
- `Kind::ConstBool` with value `false` → `falseConstant()`
- `Kind::UFApply` → intern symbol from payload name + children's sorts + node's sort, intern each child recursively, then intern application
- Arithmetic `Kind::Add/Sub/Mul/...` → return `NullEufTerm`
- Nested `Kind::Eq/Distinct` inside arguments → return `NullEufTerm`

### 3.4 `EGraph` with Term-Level Proof Graph

**Critical design:** The proof graph records edges between **original term endpoints**, not between current class representatives.

```cpp
class EGraph {
public:
    void clear();
    void addTerm(EufTermId t);

    // Merge two terms' equivalence classes.
    // Returns SortMismatch if terms have different sorts (should not happen).
    MergeStatus merge(EufTermId a, EufTermId b, const MergeReason& reason,
                      const EufTermManager& tm);

    bool same(EufTermId a, EufTermId b) const;
    EClassId find(EufTermId t) const;

    // Close congruence: repeated scan until fixpoint.
    void closeCongruence(const EufTermManager& tm);

    // Explain why two terms are in the same class.
    // Uses the term-level proof graph.
    ExplainResult explainEquality(EufTermId a, EufTermId b,
                                  const EufTermManager& tm) const;

    // Build model: one value per class
    std::unordered_map<EClassId, std::string> buildModel() const;

    size_t classCount() const;

private:
    struct EClass {
        EClassId id;
        EClassId parent;
        std::vector<EufTermId> members;
    };

    struct ProofEdge {
        EufTermId u;        // original endpoint (NOT class rep)
        EufTermId v;        // original endpoint (NOT class rep)
        MergeReason reason;
    };

    std::vector<EClass> classes_;
    std::vector<ProofEdge> proofEdges_;

    // Union-Find
    EClassId ufFind(EClassId x) const;
    void ufUnite(EClassId a, EClassId b);

    // Congruence detection
    struct AppKey {
        FuncSymbolId symbol;
        std::vector<EClassId> argClasses;
        bool operator==(const AppKey&) const = default;
    };
    struct AppKeyHash {
        std::size_t operator()(const AppKey& k) const;
    };

    std::vector<std::pair<EufTermId, EufTermId>> findCongruentPairs(
        const EufTermManager& tm) const;

    // Explanation helpers
    ExplainResult explainMergeReason(const MergeReason& reason,
                                     const EufTermManager& tm,
                                     std::unordered_set<uint64_t>& visitedPairs) const;

    ExplainResult explainPath(EufTermId from, EufTermId to,
                              const EufTermManager& tm,
                              std::unordered_set<uint64_t>& visitedPairs) const;

    static uint64_t pairKey(EufTermId a, EufTermId b) {
        if (a > b) std::swap(a, b);
        return (static_cast<uint64_t>(a) << 32) | static_cast<uint64_t>(b);
    }
};
```

**Merge algorithm:**
```cpp
MergeStatus EGraph::merge(EufTermId a, EufTermId b, const MergeReason& reason,
                          const EufTermManager& tm) {
    if (tm.node(a).sort != tm.node(b).sort) {
        return MergeStatus::SortMismatch;
    }

    if (same(a, b)) return MergeStatus::AlreadySame;

    // Record proof edge BEFORE uniting (captures original endpoints)
    proofEdges_.push_back({a, b, reason});

    EClassId ca = find(a);
    EClassId cb = find(b);
    ufUnite(ca, cb);
    return MergeStatus::Merged;
}
```

```cpp
ExplainResult EGraph::explainMergeReason(const MergeReason& reason,
                                         const EufTermManager& tm,
                                         std::unordered_set<uint64_t>& visitedPairs) const {
    if (reason.kind == MergeReasonKind::AssertedEquality) {
        return {true, {reason.assertedLit}};
    }

    if (reason.kind == MergeReasonKind::Congruence) {
        std::vector<SatLit> result;
        for (const auto& [ai, bi] : reason.argPairs) {
            auto sub = explainPath(ai, bi, tm, visitedPairs);
            if (!sub.ok) return {false, {}};
            appendUnique(result, sub.reasons);
        }
        return {true, result};
    }

    return {false, {}};  // unknown reason kind
}
```

**Congruence closure (V1 simple repeated-scan):**
```cpp
void EGraph::closeCongruence(const EufTermManager& tm) {
    bool changed = true;
    while (changed) {
        changed = false;
        auto pairs = findCongruentPairs(tm);
        for (auto [a, b] : pairs) {
            if (same(a, b)) continue;

            const auto& na = tm.node(a);
            const auto& nb = tm.node(b);

            MergeReason reason;
            reason.kind = MergeReasonKind::Congruence;
            reason.lhsApp = a;
            reason.rhsApp = b;
            for (size_t i = 0; i < na.args.size(); ++i) {
                reason.argPairs.push_back({na.args[i], nb.args[i]});
            }

            auto status = merge(a, b, reason, tm);
            if (status == MergeStatus::SortMismatch) {
                // Internal error: congruent applications should have same sort
                // This should never happen
            }
            if (status == MergeStatus::Merged) {
                changed = true;
            }
        }
    }
}
```

### 4.2 TheoryAtomRegistry: Get-or-Create EUF Atom API

**File:** `src/theory/TheoryAtomRegistry.h`

Add:
```cpp
// Returns the SAT literal for the atom (positive polarity).
// If atom already exists, returns existing literal.
// If not, creates a fresh SAT variable and registers the atom.
SatLit getOrCreateEufAtom(const EufAtomPayload& payload, ExprId originExpr);
```

The registry internally:
1. Canonicalizes the key (sort lhs/rhs by ExprId for dedup)
2. Checks `eufLookup_`
3. If found: returns `SatLit::positive(existingSatVar)`
4. If not found: calls `atomizer_->registerDynamicAtom(originExpr, TheoryId::EUF)` to get fresh `SatVar`, registers the atom, stores in lookup

**Important:** `registerDynamicAtom` returns `SatLit::positive(var)`. The registry must store the mapping from the canonical key to the `SatVar` (not the `SatLit`, since polarity is handled separately).

Add internal data structures:
```cpp
struct EufLookupKey {
    ExprId lhs;
    ExprId rhs;
    Relation rel;
    bool operator==(const EufLookupKey& o) const {
        return lhs == o.lhs && rhs == o.rhs && rel == o.rel;
    }
};
struct EufLookupKeyHash {
    std::size_t operator()(const EufLookupKey& k) const {
        std::size_t h = std::hash<ExprId>{}(k.lhs);
        h ^= std::hash<ExprId>{}(k.rhs) + 0x9e3779b9 + (h << 6) + (h >> 2);
        h ^= static_cast<std::size_t>(k.rel) + 0x9e3779b9 + (h << 6) + (h >> 2);
        return h;
    }
};
std::unordered_map<EufLookupKey, SatVar, EufLookupKeyHash> eufLookup_;
```

### 4.3 Update Atomizer to Use Get-or-Create

Instead of:
```cpp

registry_->registerParsedTheoryAtom(v, eid, TheoryId::EUF, payload);
```

```cpp
    ActiveAssignment a;
    if (!extractFromPayload(atom, value, assertedLit, a)) {
        return;
    }
    trail_.push_back(a);
}
```

### 5.3 `backtrackToLevel` (Fixed)

```cpp
void EufSolver::backtrackToLevel(int level) {
    auto it = std::remove_if(trail_.begin(), trail_.end(),
        [level](const auto& a) { return a.level > level; });
    trail_.erase(it, trail_.end());

    // Fix scope limits that may now exceed trail size
    for (auto& lim : scopeLimits_) {
        lim = std::min(lim, trail_.size());
    }
}
```

### 5.4 `push`/`pop`

```cpp
void EufSolver::push() {
    scopeLimits_.push_back(trail_.size());
}

void EufSolver::pop(uint32_t n) {
    while (n-- && !scopeLimits_.empty()) {
        trail_.resize(scopeLimits_.back());
        scopeLimits_.pop_back();
    }
}
```

### 5.5 `check()` — Full Rebuild

```cpp
TheoryCheckResult EufSolver::check(TheoryLemmaDatabase&) {
    if (!coreIr_) {
        return TheoryCheckResult::unknown();
    }

    termManager_.clear();
    egraph_.clear();
    equalities_.clear();
    disequalities_.clear();

    // 1. Build effective atoms and intern terms
    for (const auto& a : trail_) {
        Relation effRel = a.value ? a.rawRel
                        : (a.rawRel == Relation::Eq ? Relation::Neq : Relation::Eq);

        EufTermId lhs = termManager_.intern(a.lhsExpr, *coreIr_);
        EufTermId rhs = termManager_.intern(a.rhsExpr, *coreIr_);

        if (lhs == NullEufTerm || rhs == NullEufTerm) {
            return TheoryCheckResult::unknown();
        }

        if (effRel == Relation::Eq) {
            equalities_.push_back({lhs, rhs, effRel, a.assertedLit});
        } else {
            disequalities_.push_back({lhs, rhs, effRel, a.assertedLit});
        }
    }

    // 2. Register all terms in EGraph
    for (EufTermId t = 0; t < termManager_.termCount(); ++t) {
        egraph_.addTerm(t);
    }

    // 2b. Built-in: true and false are distinct
    if (termManager_.trueConstant() != NullEufTerm &&
        termManager_.falseConstant() != NullEufTerm) {
        if (egraph_.same(termManager_.trueConstant(), termManager_.falseConstant())) {
            // This should only happen if some asserted equality forced true=false
            auto er = egraph_.explainEquality(
                termManager_.trueConstant(),
                termManager_.falseConstant(),
                termManager_);
            if (!er.ok) {
                return TheoryCheckResult::unknown();
            }
            std::vector<SatLit> clause;
            for (auto lit : er.reasons) {
                clause.push_back(lit.negated());
            }
            return TheoryCheckResult::mkConflict(TheoryConflict{clause});
        }
    }

    // 3. Assert equalities
    for (const auto& eq : equalities_) {
        MergeReason reason;
        reason.kind = MergeReasonKind::AssertedEquality;
        reason.assertedLit = eq.assertedLit;
        auto status = egraph_.merge(eq.lhs, eq.rhs, reason, termManager_);
        if (status == MergeStatus::SortMismatch) {
            return TheoryCheckResult::unknown();
        }
    }

    // 4. Close congruence
    egraph_.closeCongruence(termManager_);

    // 5. Check disequalities
    for (const auto& neq : disequalities_) {
        if (egraph_.same(neq.lhs, neq.rhs)) {
            auto er = egraph_.explainEquality(neq.lhs, neq.rhs, termManager_);
            if (!er.ok) {
                return TheoryCheckResult::unknown();
            }

            std::vector<SatLit> clause;
            for (auto lit : er.reasons) {
                clause.push_back(lit.negated());
            }
            clause.push_back(neq.assertedLit.negated());

            return TheoryCheckResult::mkConflict(TheoryConflict{clause});
        }
    }

    return TheoryCheckResult::consistent();
}
```
