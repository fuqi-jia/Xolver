# EUF Solver V1 — Pure QF_UF Baseline (Revised V5)

## Goal
Implement a **sound, minimal** QF_UF theory solver. No mixed theories. No Nelson-Oppen. Full-rebuild congruence closure with term-level explanation.

## What V1 Supports
- Uninterpreted sorts, constants, and function applications
- Binary and n-ary equality / distinct (normalized to pairwise atoms)
- Bool-valued terms in formula position (UF predicates, Bool variables) → `term = true`
- Congruence closure
- Theory conflict explanation
- Full-rebuild check()

## What V1 Does NOT Support
- QF_UFLIA / QF_UFLRA / QF_UFNIA / QF_UFNRA (return Unknown)
- UF + arithmetic mixed terms
- Nelson-Oppen / equality sharing
- Theory propagation
- Incremental e-graph rollback

---

## Red Lines

1. **Only QF_UF is registered.** All mixed logics return Unknown.
2. **Unknown logics do NOT fall back to LRA.**
3. **Explanation uses term-level proof graph.** Never class-root BFS.
4. **Function symbol key = name + arg sorts + result sort.**
5. **Both true and false constants exist; true != false is enforced after congruence closure.**
6. **N-ary equality/distinct normalizes to pairwise atoms with unique synthetic ExprIds.**
7. **Atomizer owns EUF dedup via `unordered_map<EufAtomKey, SatLit>`.** Registry only records atoms.
8. **Sort mismatch returns MergeStatus::SortMismatch, not silent ignore.**
9. **Explanation cycle returns failure (ok=false), not empty reasons.**
10. **closeCongruence returns CloseStatus; caller checks for errors.**
11. **Trail-based push/pop with backtrack-safe scope limits.**
12. **`assertedLit` naming propagated to all theory solvers.**
13. **Bool sort ID obtained from SOMTParser sort system, not ConstBool scanning.**
14. **Synthetic ExprIds allocated from descending range.**

### 2.4 Atomizer EUF dedup helper

```cpp
SatLit Atomizer::getOrCreateEufAtom(EufAtomPayload payload, ExprId originExpr) {
    // Canonicalize payload in-place
    if (payload.lhsExpr > payload.rhsExpr) {
        std::swap(payload.lhsExpr, payload.rhsExpr);
    }

    EufAtomKey key{payload.lhsExpr, payload.rhsExpr, payload.rel};
    auto it = eufAtomDedup_.find(key);
    if (it != eufAtomDedup_.end()) {
        memo_[originExpr] = it->second;
        return it->second;
    }

    SatVar v = freshVar();
    SatLit lit = SatLit::positive(v);
    atoms_.push_back({v, originExpr, true, TheoryId::EUF});
    eufAtomDedup_[key] = lit;
    memo_[originExpr] = lit;

    if (registry_) {
        registry_->registerParsedTheoryAtom(v, originExpr, TheoryId::EUF, payload);
    }

    return lit;
}
```

**Registry does NOT create or dedup SAT vars.** It only records atoms for `findBySatVar`.

### 2.5 N-ary equality normalization

```cpp
SatLit Atomizer::atomizeNaryEufEq(ExprId eid, const CoreIr& ir) {
    const auto& e = ir.get(eid);
    SatVar andVar = freshVar();
    std::vector<SatLit> pairwiseLits;

    for (size_t i = 0; i + 1 < e.children.size(); ++i) {
        ExprId lhs = e.children[i];
        ExprId rhs = e.children[i + 1];
        ExprId synthExpr = synthExprAlloc_.next();
        SatLit plit = getOrCreateEufAtom(
            EufAtomPayload{lhs, rhs, Relation::Eq}, synthExpr);
        pairwiseLits.push_back(plit);
    }

    // Tseitin AND
    for (SatLit pl : pairwiseLits) {
        sat_.addClause({SatLit::negative(andVar), pl});
    }
    std::vector<SatLit> clause = {SatLit::positive(andVar)};
    for (SatLit pl : pairwiseLits) clause.push_back(pl.negated());
    sat_.addClause(clause);

    return SatLit::positive(andVar);
}
```

### 2.7 Bool-valued term handling

In `atomizeRec`, BEFORE the main switch, check if this is a Bool-valued term in formula position under QF_UF:

```cpp
SatLit Atomizer::atomizeRec(ExprId eid, const CoreIr& ir) {
    if (eid == NullExpr) return SatLit{0, true};
    auto it = memo_.find(eid);
    if (it != memo_.end()) return it->second;

    const CoreExpr& e = ir.get(eid);

    // Bool-valued terms in formula position under QF_UF:
    // p(a), q (Bool var), etc. → (= term true)
    if (defaultTheory_ == TheoryId::EUF &&
        e.sort == boolSortId_ &&
        isFormulaPositionTerm(e.kind)) {
        SatLit result = getOrCreateEufAtom(
            EufAtomPayload{eid, TrueSentinelExpr, Relation::Eq}, eid);
        memo_[eid] = result;
        return result;
    }

    // ... existing switch ...
}

bool Atomizer::isFormulaPositionTerm(Kind k) const {
    return k == Kind::Variable || k == Kind::UFApply;
}
```

**Note:** `ConstBool` is handled inside the switch (true → positive unit clause, false → negative). It is NOT a formula-position term in this context because it already has a fixed boolean value.

### Intern logic

```cpp
EufTermId EufTermManager::intern(ExprId exprId, const CoreIr& ir) {
    if (exprId == TrueSentinelExpr) return internTrueConstant();
    if (exprId == FalseSentinelExpr) return internFalseConstant();

    const auto& e = ir.get(exprId);

    if (e.kind == Kind::ConstBool) {
        bool val = std::get<bool>(e.payload.value);
        return val ? internTrueConstant() : internFalseConstant();
    }

    if (e.kind == Kind::Variable) {
        FuncSymbolKey key;
        key.name = std::get<std::string>(e.payload.value);
        key.resultSort = e.sort;
        FuncSymbolId sym = internSymbol(key);
        return createNode(sym, {}, e.sort, exprId);
    }

    if (e.kind == Kind::UFApply) {
        std::vector<EufTermId> argIds;
        for (ExprId child : e.children) {
            EufTermId tid = intern(child, ir);
            if (tid == NullEufTerm) return NullEufTerm;
            argIds.push_back(tid);
        }

        FuncSymbolKey key;
        key.name = std::get<std::string>(e.payload.value);
        key.resultSort = e.sort;
        for (EufTermId tid : argIds) {
            key.argSorts.push_back(nodes_[tid].sort);
        }
        FuncSymbolId sym = internSymbol(key);
        return createNode(sym, std::move(argIds), e.sort, exprId);
    }

    return NullEufTerm;
}
```

```cpp
                              const EufTermManager& tm,
                              std::unordered_set<uint64_t>& visitedPairs) const;

    static uint64_t pairKey(EufTermId a, EufTermId b) {
        if (a > b) std::swap(a, b);
        return (static_cast<uint64_t>(a) << 32) | static_cast<uint64_t>(b);
```

**merge:**

```cpp
MergeStatus EGraph::merge(EufTermId a, EufTermId b, const MergeReason& reason,
                          const EufTermManager& tm) {
    if (tm.node(a).sort != tm.node(b).sort) {
        return MergeStatus::SortMismatch;
    }
    if (same(a, b)) return MergeStatus::AlreadySame;

    proofEdges_.push_back({a, b, reason});
    EClassId ca = find(a);
    EClassId cb = find(b);
    ufUnite(ca, cb);
    return MergeStatus::Merged;
}
```

**closeCongruence:**
```cpp
CloseStatus EGraph::closeCongruence(const EufTermManager& tm) {
    bool changed = true;
    while (changed) {
        changed = false;
        auto pairs = findCongruentPairs(tm);
        for (auto [a, b] : pairs) {
            if (same(a, b)) continue;

            const auto& na = tm.node(a);
            const auto& nb = tm.node(b);

            MergeReason reason;
            reason.kind = MergeReasonKind::Congruence;
            reason.lhsApp = a;
            reason.rhsApp = b;
            for (size_t i = 0; i < na.args.size(); ++i) {
                reason.argPairs.push_back({na.args[i], nb.args[i]});
            }

            auto status = merge(a, b, reason, tm);
            if (status == MergeStatus::SortMismatch) {
                return CloseStatus::SortMismatch;
            }
            if (status == MergeStatus::Merged) {
                changed = true;
            }
        }
    }
    return CloseStatus::Ok;
}
```

**explainPath:**
```cpp
ExplainResult EGraph::explainPath(EufTermId from, EufTermId to,
                                  const EufTermManager& tm,
                                  std::unordered_set<uint64_t>& visitedPairs) const {
    if (from == to) return {true, {}};

    auto pk = pairKey(from, to);
    if (visitedPairs.count(pk)) {
        return {false, {}};  // cycle in explanation graph; fail
    }
    visitedPairs.insert(pk);

    // BFS on proofEdges_ (undirected)
    // Build adjacency on the fly
    // Find shortest path from 'from' to 'to'

    if (!pathFound) {
        return {false, {}};  // no path found; internal error
    }

    std::vector<SatLit> result;
    for (size_t edgeId : pathEdges) {
        const auto& edge = proofEdges_[edgeId];
        auto sub = explainMergeReason(edge.reason, tm, visitedPairs);
        if (!sub.ok) return {false, {}};
        appendUnique(result, sub.reasons);
    }
    return {true, result};
}
```

**explainMergeReason:**
```cpp
ExplainResult EGraph::explainMergeReason(const MergeReason& reason,
                                         const EufTermManager& tm,
                                         std::unordered_set<uint64_t>& visitedPairs) const {
    if (reason.kind == MergeReasonKind::AssertedEquality) {
        return {true, {reason.assertedLit}};
    }

    if (reason.kind == MergeReasonKind::Congruence) {
        std::vector<SatLit> result;
        for (const auto& [ai, bi] : reason.argPairs) {
            auto sub = explainPath(ai, bi, tm, visitedPairs);
            if (!sub.ok) return {false, {}};
            appendUnique(result, sub.reasons);
        }
        return {true, result};
    }

    return {false, {}};  // unknown reason kind
}
```

```cpp
void EufSolver::backtrackToLevel(int level) {
    auto it = std::remove_if(trail_.begin(), trail_.end(),
        [level](const auto& a) { return a.level > level; });
    trail_.erase(it, trail_.end());
    for (auto& lim : scopeLimits_) {
        lim = std::min(lim, trail_.size());
```

```cpp
        size_t lim = std::min(scopeLimits_.back(), trail_.size());
        trail_.resize(lim);
```

**check — CORRECT ORDER:**
```cpp
TheoryCheckResult EufSolver::check(TheoryLemmaDatabase&) {
    if (!coreIr_) {
        return TheoryCheckResult::unknown();
    }

    termManager_.clear();
    egraph_.clear();
    equalities_.clear();
    disequalities_.clear();

    // 1. Build effective atoms and intern terms
    for (const auto& a : trail_) {
        Relation effRel = a.value ? a.rawRel
                        : (a.rawRel == Relation::Eq ? Relation::Neq : Relation::Eq);

        EufTermId lhs = termManager_.intern(a.lhsExpr, *coreIr_);
        EufTermId rhs = termManager_.intern(a.rhsExpr, *coreIr_);

        if (lhs == NullEufTerm || rhs == NullEufTerm) {
            return TheoryCheckResult::unknown();
        }

        if (effRel == Relation::Eq) {
            equalities_.push_back({lhs, rhs, effRel, a.assertedLit});
        } else {
            disequalities_.push_back({lhs, rhs, effRel, a.assertedLit});
        }
    }

    // 2. Register all terms in EGraph
    for (EufTermId t = 0; t < termManager_.termCount(); ++t) {
        egraph_.addTerm(t);
    }

    // 3. Assert equalities
    for (const auto& eq : equalities_) {
        MergeReason reason;
        reason.kind = MergeReasonKind::AssertedEquality;
        reason.assertedLit = eq.assertedLit;
        auto status = egraph_.merge(eq.lhs, eq.rhs, reason, termManager_);
        if (status == MergeStatus::SortMismatch) {
            return TheoryCheckResult::unknown();
        }
    }

    // 4. Close congruence
    auto closeStatus = egraph_.closeCongruence(termManager_);
    if (closeStatus != CloseStatus::Ok) {
        return TheoryCheckResult::unknown();
    }

    // 5. Built-in: true and false are distinct
    if (termManager_.trueConstant() != NullEufTerm &&
        termManager_.falseConstant() != NullEufTerm &&
        egraph_.same(termManager_.trueConstant(), termManager_.falseConstant())) {

        auto er = egraph_.explainEquality(
            termManager_.trueConstant(),
            termManager_.falseConstant(),
            termManager_);
        if (!er.ok) {
            return TheoryCheckResult::unknown();
        }

        std::vector<SatLit> clause;
        for (auto lit : er.reasons) {
            clause.push_back(lit.negated());
        }
        return TheoryCheckResult::mkConflict(TheoryConflict{clause});
    }

    // 6. Check disequalities
    for (const auto& neq : disequalities_) {
        if (egraph_.same(neq.lhs, neq.rhs)) {
            auto er = egraph_.explainEquality(neq.lhs, neq.rhs, termManager_);
            if (!er.ok) {
                return TheoryCheckResult::unknown();
            }

            std::vector<SatLit> clause;
            for (auto lit : er.reasons) {
                clause.push_back(lit.negated());
            }
            clause.push_back(neq.assertedLit.negated());

            return TheoryCheckResult::mkConflict(TheoryConflict{clause});
        }
    }

    return TheoryCheckResult::consistent();
}
```
